# FlowyGraphSample

这是一个关于 **FlowyGraph** 插件的官方示例项目。

### 项目简介

**FlowyGraph** 是一款专为 Unity 开发的**面向策划工作的节点式叙事逻辑插件**。它通过可视化的方式，帮助开发者和策划快速构建游戏中的叙事逻辑、对话分支、任务流程等复杂交互。

### 核心特性

- **逻辑可视化**：提供直观的节点图编辑器，使叙事分支和逻辑流向一目了然。
- **自定义节点**：支持低代码扩展，程序可以轻松编写自定义节点供策划使用。
- **团队工作解耦**：策划使用节点图制作流程，程序专注于节点功能的具体实现。
- **信号通信系统**：支持图内/图外的异步事件通信，处理复杂的交互反馈。
